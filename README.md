# 🎬 Anime Streaming - Front-End Project

Este projeto é um **site de streaming de animes** desenvolvido para praticar **HTML, CSS e JavaScript**, simulando a interface de uma plataforma de streaming.

## ⚠️ Aviso Importante

Devido às limitações de tamanho de arquivo do GitHub (máximo de **100MB por arquivo**), não foi possível enviar todos os vídeos do projeto para o repositório.

Por esse motivo, **apenas alguns animes estão funcionando no player os outros o player ate funciona mas nao abre o video**.

## ✅ Animes disponíveis no player

Atualmente o player está funcionando para os seguintes animes:

* Eighty Six (86)
* Demon Slayer
* sword art online
* Darling in the Franxx

Esses foram mantidos apenas para demonstrar o funcionamento do **sistema de player e navegação entre episódios**.

## 💻 Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript

## 🎯 Objetivo do projeto

Este projeto foi criado com fins de **estudo e portfólio**, com o objetivo de demonstrar:

* criação de interface estilo streaming
* manipulação do player de vídeo
* organização de conteúdo em cards
* navegação entre páginas
* uso de JavaScript para controle de episódios

## 🚀 Observação

Mesmo com a limitação de arquivos de vídeo, o projeto demonstra o funcionamento completo da interface de streaming e da lógica do player.
